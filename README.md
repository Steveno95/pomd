# pomd
[![Standard - JavaScript Style Guide](https://cdn.rawgit.com/feross/standard/master/badge.svg)](https://github.com/feross/standard)

üçÖ Just a good old cli based pomodoro timer with  native notifactions.

## Install

```
$ npm install -g pomd
```

## Usage

1. Run `pomd` in your terminal.
2. Run `help` to see the options available.

## example 

```
$ pomd
üçÖ start --time 20:00 --chill 03:00
```
This will start a 20 minute session with 3 minute break
